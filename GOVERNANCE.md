# Zarządzanie projektem

Placówka Online jest projektem utrzymywanym przez Adama Trojanowskiego.
Opiekun zatwierdza wydania, zmiany bezpieczeństwa, kierunek projektu oraz użycie
certyfikatu Code Signing. Decyzje techniczne powinny być udokumentowane w Issue,
Pull Request lub pliku ADR w `docs/adr`.

Wkład społeczności jest mile widziany, ale włączenie zmiany zależy od jakości,
bezpieczeństwa, zgodności z celem projektu i możliwości utrzymania jej w czasie.
