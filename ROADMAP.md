# Plan rozwoju

## Stabilizacja

- testy API i workflow incydentów;
- pełna dokumentacja odtwarzania kopii;
- automatyczny SBOM i kontrola zależności;
- dopracowanie aktualizacji istniejących agentów.

## Rozwój monitoringu

- wybrane zdarzenia Windows Event Log;
- certyfikaty TLS i terminy wygaśnięcia;
- stan kopii zapasowych na urządzeniach;
- kontrola aktualności agenta i bezpieczny kanał aktualizacji.

Mapa sieci i agresywne skanowanie podsieci nie należą do bieżącego zakresu.
