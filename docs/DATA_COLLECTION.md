# Zakres zbieranych danych

Agent zbiera wyłącznie dane techniczne potrzebne do monitoringu, m.in.:

- identyfikację urządzenia i wersję systemu;
- czas pracy, CPU, RAM i pojemność dysków;
- SMART;
- podstawowe dane kart sieciowych, IP, bramy i opóźnień;
- dostępność Internetu, DNS i serwera;
- stan wybranych usług Windows;
- Windows Update i wymagany restart;
- stan dostawcy ochrony z Windows Security Center;
- stan Zapory Windows;
- wersję i samokontrolę agenta.

Agent nie jest przeznaczony do zbierania dokumentów, treści poczty, historii
przeglądania, haseł, naciśnięć klawiatury, schowka, obrazu ekranu, dźwięku ani
obrazu z kamery.

Instalacja agenta wymaga zgody podmiotu zarządzającego urządzeniem.
