# Oficjalne adresy repozytorium

## Strona WWW

```text
https://github.com/wcagcms/placowka-online
```

## Git przez HTTPS

```text
https://github.com/wcagcms/placowka-online.git
```

## Git przez SSH

```text
git@github.com:wcagcms/placowka-online.git
```

## Zmiana istniejącego `origin`

```bash
git remote set-url origin https://github.com/wcagcms/placowka-online.git
git remote -v
```

W dokumentacji, plikach konfiguracyjnych i publicznych stronach prawnych należy
stosować adres bez końcówki `.git`: `https://github.com/wcagcms/placowka-online`.
