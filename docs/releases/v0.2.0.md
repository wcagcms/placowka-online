# Placówka Online 0.2.0

Pierwsze wydanie publikowane w organizacji **wcagcms**.

## Najważniejsze zmiany

- agent Windows `exe-1.9.3`;
- instalator `1.0.6`;
- poprawna sonda Microsoft NCSI przez HTTP;
- odporność na pojedyncze awarie zewnętrznych sond;
- incydent Internetu otwierany po 3 kolejnych błędach;
- incydent zamykany po 5 kolejnych prawidłowych pomiarach;
- jedna wiadomość przy otwarciu i jedna przy zamknięciu tego samego incydentu;
- obsługa Microsoft Defender, ESET i innych dostawców ochrony;
- Secure Enrollment, bufor offline, kopie zapasowe i workflow incydentów.
