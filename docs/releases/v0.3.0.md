# Placówka Online v0.3.0 — Security Hardening

Wydanie wzmacnia zabezpieczenia panelu i procesu publikacji bez zmiany protokołu agenta.

## Najważniejsze zmiany

- zaufane wartości nagłówka `Host`;
- pełniejsza Content Security Policy bez skryptów inline;
- szyfrowanie sesji i bezpieczne cookies jako domyślne ustawienia produkcyjne;
- polecenie `placowka:security-check`;
- plik `.env.production.example`;
- testy uwierzytelniania, izolacji placówek, heartbeat i nagłówków HTTP;
- audyt Composer i npm w CI;
- przypięcie `checkout` i `setup-node` do niezmiennych commitów;
- kontrola podpisów wszystkich plików EXE wydania;
- lista ustawień bezpieczeństwa GitHub.

## Zgodność

- migracja bazy: nie;
- aktualizacja agenta: nie;
- nowy instalator: nie;
- ponowna rejestracja urządzeń: nie;
- wymagane wylogowanie operatorów po wdrożeniu: tak.
