# Opis projektu na potrzeby Certum

Placówka Online jest nieodpłatnym projektem open source przeznaczonym dla
jednostek samorządu terytorialnego oraz podległych im placówek. System służy do
monitorowania stanu technicznego komputerów, wykrywania awarii i wspierania
ciągłości pracy infrastruktury IT.

Panel projektu: https://monitoring.wcag-cms.pl/panel/login

Repozytorium: https://github.com/wcagcms/placowka-online

Kod obejmuje aplikację Laravel, agent Windows w Go, pomocnik bezpiecznej
rejestracji, definicję instalatora Inno Setup i skrypty budowania.

Certyfikat Open Source Code Signing będzie używany wyłącznie do podpisywania
oficjalnego instalatora i plików wykonywalnych Placówka Online.


Aktualna wersja źródeł publicznych: `v0.2.0`.

Wersja agenta: `exe-1.9.3`.

Wersja instalatora: `1.0.6`.
