# Deklaracja wykorzystania certyfikatu

Ja, Adam Trojanowski, oświadczam, że certyfikat Open Source Code Signing będzie
wykorzystywany wyłącznie do podpisywania oficjalnych plików projektu Placówka
Online.

Projekt jest rozwijany jako open source na licencji AGPL-3.0-or-later i
udostępniany nieodpłatnie jednostkom samorządu terytorialnego oraz podległym im
placówkom.

Agent nie jest przeznaczony do ukrytego monitorowania osób. Nie zawiera funkcji
keyloggera, przechwytywania haseł, zrzutów ekranu ani odczytu prywatnych
dokumentów. Zbiera dane techniczne opisane w `docs/DATA_COLLECTION.md`.

Podpisywane pliki:

- PlacowkaOnlineSetup.exe;
- PlacowkaOnlineAgent.exe;
- PlacowkaOnlineAgentConsole.exe;
- PlacowkaOnlineEnroll.exe.
